# Hands-On 6 – SQLAlchemy ORM

## Objective
Learn ORM concepts using SQLAlchemy with MySQL.

## Files
- models.py
- crud.py
- loading_demo.py
- eager_loading.py
- requirements.txt

## Topics Covered
- ORM Models
- Relationships
- CRUD Operations
- Lazy Loading
- Eager Loading
- N+1 Query Problem

## Run

```bash
pip install -r requirements.txt
python models.py
python crud.py
python loading_demo.py
python eager_loading.py
```
